# Spiders

https://pi.math.cornell.edu/~hubbard/SpidersFinal.pdf

This figure from [TreesBook](https://www.mat.univie.ac.at/%7Ebruin/talks/TreesBook.pdf) shows many of the descriptions of quadratic maps.
